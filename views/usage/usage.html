Usage of this app:

This app is to be used to help test, explore and understand various document viewing services. At the moment, PDFTron is the only service being tested.

  1. Begin by going the the PDFTron tab. Click the "Preview" button, and a popop should be displayed, showing an XOD document inside of a PDFTron.WebViewer instance.

The div in which the PDFTron WebViewer is contained in is named with the CSS id, "NewtonDocumentViewer". This sample app is responsive, and can be used to test how well the iframe in the mobile version of PDFTron's MobileReaderControl sits within the "NewtonDocumentViewer" container.

Feel free to enhance and change the application as desired.
